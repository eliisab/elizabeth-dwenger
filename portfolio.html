Coding Journey

First Steps

I created this project when I first started self-teaching myself HTML and CSS. My main focus with this project was to make my first webpage with a minimalistic design which could fit well for mobile viewing; this website is about the Dutch art movement, De Stijl.

C++ Sorter

In terms of coding and computer science, this project is probably one of the more difficult things I've tackled. Not only is it in C++, a language which I am less familiar with, it also deals with pointers and search algorithms. This project taught me a lot about these aforementioned topics and is a project I am very proud of (however, the stress that it caused me makes me hesitant to tackle such a coding problem on my own again, at least in C++).

Personal Portfolio

Using the principals of HTML and CSS which I taught myself, I decided to make this portfolio that you're looking at now. This was mostly done in my freetime, and incorperates a lot of inspiration from contemperary graphic design & typography portfolios. When thinking about how to apply what I had learned about HTML and CSS further, the obvious choice was to create a portfolio with my CV & photography work.

Animal Shelter Website

When I was still living in Minnesota, I volunteered at an animal shelter near my high school for six years. The webpage which the animal shelter uses is very outdated in terms of design, so as a personal project, I spent four weekends creating this website. It is only a couple of webpages, but it focuses on design and minimalism to create a better experience for the user.

Design for Mobile

For a group project about UX and UI design within my Human Computer Interaction course, my groupmates and I decided to design a diet app. This project had the focus of using Scandinavian minimalism to create an app which would be easy to use for an audience of people who wished to lose weight. I took responsibility to create the wireframe and prototype of this app with the use of AdobeXD.
